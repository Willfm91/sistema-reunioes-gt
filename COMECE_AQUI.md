## ⚡ PRIMEIRO: Obter sua Chave de API Claude

1. Acesse https://console.anthropic.com/account/keys
2. Clique em "Create Key"
3. Copie a chave (começa com `sk-`)
4. **Guarde bem** - você vai usar no Vercel

---

## 🚀 SEGUNDO: Fazer Deploy em 5 Minutos

Siga EXATAMENTE o arquivo `DEPLOY_GUIDE.md` - é super direto!

Ele tem:
- ✅ Passo 1: Criar repo no GitHub (2 min)
- ✅ Passo 2: Deploy no Vercel (2 min)
- ✅ Passo 3: Adicionar chave de API (1 min)
- ✅ Passo 4: Clicar Deploy (30 seg)

---

## ✅ TERCEIRO: Testar Funcionamento

Depois que o Vercel terminar o deploy:

1. Você vai receber um link tipo:
   ```
   https://seu-projeto.vercel.app
   ```

2. Abra o link no navegador

3. Testa:
   - Clique em "⚙️ Processar"
   - Clique em "🔗 Conectar Google Drive"
   - Cole uma transcrição qualquer
   - Clique em "🚀 Processar"
   - Se mostrar resultado = ✅ FUNCIONANDO!

## 🤔 Dúvidas?

**P: Onde coloco minha chave de API?**
- R: No Vercel, no campo "Environment Variables" durante o deploy
- Leia "DEPLOY_GUIDE.md" - PASSO 3

**P: Posso testar no meu computador antes?**
- R: Sim! Mas não é necessário. O Vercel testa automaticamente.
- Se quiser: siga em `README.md` → seção "Desenvolvimento Local"

**P: E depois que está online?**
- R: Toda vez que você faz `git push`, Vercel faz deploy automático

**P: Meus dados são seguros?**
- R: Sim! Ficam no seu navegador, não em servidor
